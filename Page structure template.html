Basic Information

Semantic Elements

Semantic Elements include Header, Nav, Section, Article, Main, and Footer.

Why use semantic HTML

There are several reasons for using semantic HTML tags with your web page. It helps with organization for your page. It allows for better collaboration when multiple people are working on the same site. It better provides information about your page to browsers and search engines. Styling is even made easier when semantic HTML is consitanly used.

Alternatives

There are times when it might be neccessary to use tags for containers that are not semantic HTML.

A div is a nonsermantic tag that can be specified by using class or id's to go along with it.
